UPDATE KOST AJ LANRAKI

Cara pasang:
1. Ekstrak ZIP ini.
2. Copy folder app, database, resources, dan routes ke folder project utama.
3. Pilih Replace / Overwrite jika diminta.
4. Jalankan:
   php artisan optimize:clear
   php artisan migrate:fresh --seed
   php artisan serve
