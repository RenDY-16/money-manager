UPDATE 3 - Kost AJ Lanraki

Fitur baru dan perbaikan:
1. Status pembayaran lunas/belum lunas bulan berjalan.
2. Tombol Terapkan Filter pada menu Data Kamar, Data Penghuni, Pemasukan, Pengeluaran, dan Laporan Keuangan.
3. Profile Admin sekarang bisa mengubah nama, email, password, dan foto profil.
4. Distribusi pengeluaran ikut tampil saat cetak laporan karena ditambahkan tabel distribusi selain grafik.
5. Saldo bersih laporan tidak lagi menjadi negatif hanya karena filter jenis pengeluaran dipilih. Ringkasan saldo memakai total pemasukan dan pengeluaran pada periode yang sama.
6. Tampilan data tabel dibuat lebih mudah dibaca dengan table-scroll, badge status, dan detail kamar/tagihan.
7. Tema gelap memakai warna hitam solid dan warna sekunder putih bersih.
8. Menu Backup Data ditambahkan di sidebar.
9. Backup data dapat diunduh dalam format JSON.
10. Tombol Export Excel ditambahkan pada Laporan Keuangan.

Cara pasang:
1. Ekstrak ZIP.
2. Copy folder app, database, resources, routes, dan file .gitignore ke project utama.
3. Pilih Replace/Overwrite.
4. Jalankan:
   php artisan optimize:clear
   php artisan migrate
   php artisan serve

Catatan:
- Tidak perlu migrate:fresh. Cukup php artisan migrate karena hanya menambah kolom profile_photo pada tabel users.
- ZIP ini tidak membawa database.sqlite agar data lokal/Railway tidak tertimpa.
- Jika di Railway, jalankan migrate melalui Railway pre-deploy command atau terminal Railway.
- Export Excel memakai format .xls berbasis tabel HTML agar tidak perlu package tambahan.
